# WebSecure

A web-based security scanner built with Python and React. WebSecure automates common security checks against a target URL or IP address and displays the results in a clean, modern interface.

## Tech Stack

**Backend**
- Python 3.14
- FastAPI + Uvicorn
- SQLAlchemy + SQLite
- Nmap, SQLMap, SSLyze, WebTech

**Frontend**
- React + Vite


## Key Features

- Port scanning with severity levels via Nmap
- SQL injection detection via SQLMap
- SSL/TLS certificate analysis via SSLyze
- Technology fingerprinting via WebTech
- Scan results saved to SQLite database
- Dark and light theme toggle
- Glassmorphism UI with animated background
